## 🚀 Installation

### One-Command Deploy (Ubuntu VPS)
```bash
bash deploy.sh
```

That's it. The script:
1. Creates `/root/nagual/` directory structure
2. Installs Python venv + dependencies
3. Writes `config.env` (edit with your API keys)
4. Deploys `core.py` (~5000 lines of production Python)
5. Embeds `SOUL.md` identity DNA
6. Creates systemd service
7. Starts the agent

### Requirements
- Ubuntu 20.04+ (tested on 24.04)
- Python 3.11+
- 2GB+ RAM
- Root access

### After Deploy
1. Edit `/root/nagual/config.env` with your API keys
2. Open `http://YOUR_IP:8000` for the dashboard
3. Talk to Nagual via Telegram (set TG_TOKEN + TG_CHAT_ID)
4. Watch it come alive 🌀
